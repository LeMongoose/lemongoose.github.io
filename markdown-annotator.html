Annotator for Markdown

======================

Rationale

When exporting from Jupyter (.ipynb) to markdown no metadata is added to the file. There may be a way to add this to the workflow of nbconvert, but I can't see it. Pelican requires Title, date etc. so rather than editing the file direct I wrote a Python script to open the designated files and the metadata.

Peculiarities

As the file is edited inline and does not checking, if the annotator is run twice for the same file it adds the metadata twice

Usage:

python annotator.py

Code

'''python from datetime import datetime

articles = { "annotator.md" : { "Title" : "An Annotator for Markdown Files", "Category" : "Code", "Tags" : "code, jupyter, markdown", "Slug": "markdown-annotator", "Authors" : "Calum Cookson", "Summary" : "File for adding meta data to start of Markdown File" } }

def add_meta_data(data, metadata): meta = "" for k,v in metadata.items(): meta += k + ":" + v + "\n" return meta + "\n" + data

for k,v in articles.items(): with open(k,"r+") as f: data = f.read() f.seek(0) v['Date'] = datetime.now().strftime("%d/%m/%Y %H:%M:%S") f.write(add_meta_data(data, v)) f.flush()

'''
